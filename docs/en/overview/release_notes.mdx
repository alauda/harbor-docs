# Release Notes

::: tip
The maintenance period for version **v2.12.z**  is from April 8, 2025 to April 8, 2026.
:::

## Compatibility and support matrix

The following table shows the compatibility and support matrix between the Alauda Build of Harbor operator and ACP versions.

| Alauda Build of Harbor Operator Version | Harbor Version | ACP Version |
| --------------------------------------- | -------------- | ----------- |
| v2.12.10                                | v2.12.3        | v4.0        |
| v2.12.4                                 | v2.12.0        | v4.0        |

## v2.12.10

### Features and Enhancements

#### Harbor v2.12.3 Upgrade

With this release, the underlying Harbor version has been upgraded to v2.12.3, bringing enhanced stability and bug fixes. For more details, please refer to the [Harbor v2.12.3 Release Notes](https://github.com/goharbor/harbor/releases/tag/v2.12.3).

#### Template Capability Enhancements

With this release, the built-in `Quick Start` and `High-Availability` deployment templates now support Redis Sentinel mode. For configuration details, please refer to [Redis Credentials](../install/02_harbor_credential.md).

#### Pod Security Policy Guidance

With this release, documentation has been added to clarify the impact of pod security policies on Harbor deployment. For more details, please refer to [Pod Security Policy](../install/03_harbor_deploy.md#pod-security-policy).

## v2.12.4

### Features and Enhancements

#### Alauda Build of Harbor

Includes Alauda Build of Harbor Operator that deploys Harbor Community Edition.

Includes deployment using the following templates:

- Quick Start: A template for quickly creating a Harbor instance, suitable for development and testing scenarios, not recommended for production environments.
- Production: A template for creating a production-ready Harbor instance, recommended for production environments.

To know more about Harbor v2.12 features, please visit [Harbor v2.12 Release Notes](https://github.com/goharbor/harbor/releases/tag/v2.12.0).
